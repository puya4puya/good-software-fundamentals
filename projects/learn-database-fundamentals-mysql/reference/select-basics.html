Bazele SELECT

Comprimat din Lecția 1. Printeaz-o sau ține-o deschisă cât timp scrii query-uri.

Forma

SELECT coloana1, coloana2
FROM nume_tabel
WHERE conditie;

SELECTCe coloane? (* = toate)
FROMDin ce tabel?
